# Вычислитель отличий

«Вычислитель отличий » — утилита для поиска отличий в конфигурационных файлах.


### Установка: ```$ npm link```

### Запуск ```$ gendiff [options] <filePath1> <filePath2>```

### Справка ```$ gendiff -h```
